StartClean Privacy Policy

Last updated: November 8th, 2025

StartClean does not collect, transmit, or sell any personal or usage data. All settings and configuration entries you create remain stored locally in your browser via chrome.storage.local and are never uploaded to any server.

The extension only inspects the titles and URLs of tabs that open during browser startup in order to decide whether they match your locally defined first-run rules. That analysis happens entirely on your device, and no browsing data leaves your machine.

If you have questions about this policy, reach out using the contact address listed for StartClean on the Chrome Web Store.

The full source code is available on GitHub so you can inspect exactly how the extension works before installing it.
